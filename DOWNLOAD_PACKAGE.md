# Gói tải về

Mình đã tạo sẵn thư mục để bạn tải project tại:

- `release/goiymonan/` (thư mục source đầy đủ)
- `release/goiymonan.zip` (file nén để tải nhanh)

## Ấn vào đâu để tải?

Trong giao diện file browser của môi trường này:

1. Mở thư mục `release`.
2. **Ấn vào file `goiymonan.zip`**.
3. Chọn **Download**.

Nếu chưa thấy, bấm refresh cây thư mục rồi vào lại `release/goiymonan.zip`.
